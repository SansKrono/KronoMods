Character Template ReadMe

Metadata.xml

Your almost there!

The first thing you want to do is rename the folder to whatever you want for your mod.

  • NO SPACES
  • Use underscores _ for spaces
  • only lowercase characters
Note Loading order is entirely based on alphabetical ordering. Keep this in mind if your using another mod.

Now open the metadata.xml and rename the <directory> line with whatever you named the folder
DO NOT CHANGE ANYTHING ELSE

once done save and close the metadata.xml file. now to actually update and do the rest, go to your isaac install directory...

go to the tools folder then ModUploader folder and open the ModUploader.exe

once open click the "Choose Mod..." browse to your mod folder and choose the metadata.xml

now you can change everything to your hearts galore. don't hit upload or worry about the thumbnail image, or "change notes" section yet. once your done close the uploader, it will save what you did.
