Player Name Image

In this section we will be creating the image that shows up when you enter a boss arena, and similar screens.
to create the base for this image you can use the Isaac Text Tool

The file that we are creating can be found in .\resources\gfx\ui\boss

Keep an original copy of this file, you will need it later!

The image created by the website will be much too big with the text much too small, but at least has the right font.
Open your preferred image editor and do these steps.
  1. Move the text to roughly in the center of the image instead of the top left corner
  2. Do a transform on the text, and increase its scale by 200% on both the X and Y
  3. Using a fill tool/command change the color of the text to #c7b299
  4. If your editor has the tools, true align the text to be centered on both the X and Y
  5. Crop the image to be exactly 192 x 64, This is the image size the game uses for the playername
  6. Save the file as playername_charactername.png (lowercase the name)

Repeat this step for your tainted character if they have a different name.
