Why is my image appearing as a messed up / black square?

This is because you need to save your image as a 32-bit (8bpc) png.

Is there any way to make my code easier to read?

Yes! Use a code editor to edit your code. A good free one is Notepad++

How do I get to where I have installed Isaac?

If you don't know how to get to your Isaac install, do this.

  1. Open Steam
  2. Right click the game
  3. Go down to Manage
  4. Click Browse Local Files
