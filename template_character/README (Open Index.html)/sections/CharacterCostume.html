Character Template ReadMe

Character Skin

This section is about adding the base costume to your character (the characters' skin)

the file can be found in .\resources\gfx\characters\costumes

It's highly recommended to use my template and just edit it instead of trying to make your own!

If you do choose to make your own (you insane person you). The file needs to be a 512x512
Save the file as character_charactername.png (lowercase the name).

Do this again for your tainted character if you want them to look different.
Save the file the same way as above if named differently.
If named the same, just add "_b" to the end of the file.

Character Costume

This section is for creating an overlay costume for the character (like maggie's hair, or cain's eyepatch)

Included in this template is a simple cat ears costume. along with its PSD reference I used to create it.
This template mod allows you to apply a costume for both the default and tainted character separately.
You must do the different file types in the same place I put them!
anm2s in /character, pngs in /character/costume.

Creating a costume takes 4 steps.
  1. Make the costume frames in .\resources\gfx\characters\costumes
    2. - you need 8 frames total, 1 for each each facing direction + 1 for each direction while firing
  2. Setup the anm2 in .\resources\gfx\characters
    3. - you can use the animation editor in the tools folder of the Isaac install (This is covered in Character menu)
  3. Tell the game your costume exists by adding it to costumes2.xml found in .\content
  4. Apply the costume in the stats.lua file

    5. If you do not have / want a costume, you can delete these files:

    • costumes2.xml
    • character_alpha_cat_ears.anm2
    • character_cat_ears_template.png

    Regardless if you do a costume or not be sure to delete the character_cat_ears_template_reference.psd file when your done using it (if you do).
