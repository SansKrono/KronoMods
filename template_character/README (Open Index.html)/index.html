Character Template ReadMe

Intro

Welcome to the "Template Character Mod"

This is the ReadMe page. It will guide you through each step of what you need to do. As you go through this guide you may also find extra comments and instructions inside some of the files to further help you.

This guide is sorted into sections. The different sections can be found on the navigation bar on the left. The sections are not necessarily sorted in order they should be or could be done, but rather are just done in a way that keeps what you need to do organized and sorted.

Before you begin you will want to come up with a name for both your character and tainted character. The tainted character can have the same name as the regular character, just know it before you get started. It will change how you do some things.
