Character Menu

In this section we will be creating the main menu icon and text

The file we will need to change is in \content\gfx
Open charactermenu.png with your editor of choice.

What you need to put in here is the picture of your character, and your character's name.
Remember that file I told you to keep an original copy of? you can use it here
The color the website produced is actually wrong for this file, but by an amount you'd be hard pressed to be able to see.
Still if you want to make it correct, change the text color to #372b2d

You will also need to do this for your tainted character; but, in charactermenualt.png

Animation Editor

DEPSITE THIS BEING NEXT I SUGGEST DOING THE FIRST PART OF Death Portraits FIRST

Now we will need to link our character to the image we just created.
To do this you will need to open the Animation Editor.
You can find the Animation Editor in the tools folder of your Isaac install.

My suggestion is to associate .anm2 files to this editor so its easier to open them in the future.
To do this you will need to open the editor in Administrator.
Go up to settings, and click the "Associate ..." line.

you can then close the application if you wish, and just go to the folder where the anm2 file is and open them directly.
if you do so, skip over this next line.

Once the editor is open, go up to File, and click Open. then you will need to go .\content\gfx

open charactermenu.anm2

On the right side of the program you will see the animations that exist within this anm2.
what you need to do is click on Alpha, hit duplicate, and name it whatever you character's name is in players.xml (including caps)
Then delete Alpha.

With your character selected on the left side you should see the bottom panel called the "timeline".
when you click one of those yellow squares with a dot it will let you edit that frame of the animation.
the only frame you Need to edit is the "Name" frame. just ensure your characters name is selected with the red bounding box.

Once your done save the file (Ctrl+S or File->Save). It will warn you about overwriting an existing file. hit accept.
for your tainted character you will need to do the same thing but in the charactermenualt.anm2 file

This same process is repeated for characterportraits.anm2 and characterportraitsalt.anm2, only the frame there you need to edit is the Portrait.
