Stage Portrait Image

In this section, we will create the player portrait used when changing floors or when you enter a boss arena.

the file we are creating can be found in ".\resources\gfx\ui\stage"

You can do this however you would like, but I would suggest creating a copy of my files and editing them from there.
If you do choose to do it by hand, the file needs to be 144 x 144 in size.
Once you've got the image, save it as playerportrait_charactername.png (lowercase the name).
For your tainted image, if they have a different name, just do what was done above. If they have the same name, just add "_b" to the end of the file name.

Repeat this step for your tainted character if they are different.
